# Belgian Financial Intelligence

Local-network dashboard — 10-year financial history for Belgian company administrators.

**Stack:** Node.js/Express · Vite/React · CBE API · NBB CBSO API

## Quick start (Mac mini)

1. Clone this repo to `~/belgian-finance`
2. Copy `backend/.env.example` → `backend/.env` and fill in your API keys
3. Open Claude Code in the repo root and run `/project:deploy-setup`
